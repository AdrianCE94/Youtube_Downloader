用户您好，欢迎来到这个程序。 该软件允许您单独或作为播放列表下载 YouTube 内容。


在“语言”部分，我们可以选择程序的语言。 我们可以放置以下内容：
+ es_ES = 西班牙语
+ en_US = 英语
+ zh_CN = 中文
+ ru_RU = 俄语
+ fr_FR = 法语（法国）
+ ar_EG = 埃及阿拉伯语（与标准最相似）
+ ja_JP = 日语
+ de_DE = 德语（德国）
+ 世界语 = 世界语

在“退出”部分，您可以关闭所有程序窗口。

在“更多信息”部分，我们可以找到以下内容：
     +“作者信息”：这指定了软件的原始创建者。
     +“贡献者信息”：在本节中，我们可以找到有关项目合作人员的信息。
     +“帮助”：我们可以找到有关程序运行的信息。
     + “关于本软件”：向我们提供有关本软件的一般信息。

在“工具-附加”部分，我们可以找到以下内容：
     +“下载单个视频”：允许我们从 YouTube 网址下载视频并将其保存在系统文件夹中。
     +“下载视频播放列表”：它允许我们以视频格式从 YouTube 播放列表下载视频。
     +“下载单个 .mp3 文件”：允许我们通过提供 url 从 YouTube 下载 mp3 格式的音乐。
     +“下载音乐播放列表”：它允许我们下载 mp3 格式的 YouTube 播放列表。
