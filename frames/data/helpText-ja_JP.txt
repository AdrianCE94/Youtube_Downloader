こんにちはユーザー、このプログラムへようこそ。 このソフトウェアを使用すると、YouTube コンテンツを個別に、またはプレイリストとしてダウンロードできます。


「言語」セクションでは、プログラムの言語を選択できます。 以下を配置できます。
+ es_ES = スペイン語
+ en_US = 英語
+zh_CN = 中国語
+ ru_RU = ロシア語
+ fr_FR = フランス語 (フランス)
+ ar_EG = アラビア語のエジプト (標準に最も類似)
+ ja_JP = 日本語
+ de_DE = ドイツ語(ドイツ)
+ エスペラント = エスペラント

「終了」セクションでは、すべてのプログラム ウィンドウを閉じることができます。

「詳細情報」セクションでは、次の情報を見つけることができます。
     + 「作成者情報」: ソフトウェアの元の作成者を指定します。
     + 「貢献者の情報」: このセクションでは、プロジェクトに協力した人々に関する情報を見つけます。
     + 「ヘルプ」: プログラムの操作に関する情報を見つけることができます。
     + 「このソフトウェアについて」: ソフトウェアに関する一般的な情報を提供します。

「Tools-Extras」セクションには、次のものがあります。
     + 「単一のビデオをダウンロード」: YouTube の URL からビデオをダウンロードし、システム フォルダーに保存できます。
     + 「動画のプレイリストをダウンロード」: YouTube プレイリストから動画を動画形式でダウンロードできます。
     + 「単一の .mp3 ファイルをダウンロード」: URL を指定して、YouTube から音楽を mp3 形式でダウンロードできます。
     + 「音楽プレイリストをダウンロード」: YouTube プレイリストを mp3 形式でダウンロードできます。
